Project Information

1 Dataset
The dataset1 used in this project contains 25,000 color images in .jpeg format, of which 10,000 are related to adenocarcinomas and benign colon tissues and 15,000 are related to lung adenocarcinomas, lung squamous cell carcinomas, and benign lung tissues. In detail, the dataset contains 750 lung tissue images, including 250 healthy lung tissue images, 250 lung adenocarcinoma images, and 250 squamous cell carcinoma images, complying with HIPAA regulations. All images were subsequently cropped using the Python programming language, resulting in square dimensions of 768x768 pixels from the original 1024x768 pixels. Subsequently, the images were augmented using the Augmentor software package, which allowed an expansion of the dataset to 15,000 images, through the following augmentations: left and right rotations (up to 25 degrees) and horizontal and vertical flips.

2 Classification

To perform the classification, a Deep Learning architecture called Standard CNN was used, composed of a convolutional layer, a pooling layer, and a fully connected layer.

2.1 Training Phase

The model was trained with the following hyperparameters:
epochs = 50; batch_size = 32; learning_rate = 0.0001; size_img = 110x3; Execution time = 1:00:17 (HH:MM:SS) Division of the dataset into 80-10-10 (training-validation-test)
